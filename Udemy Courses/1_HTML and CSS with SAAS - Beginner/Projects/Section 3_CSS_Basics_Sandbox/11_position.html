    Positioning

  1. Position by default is 'static'.



  2. Relative Positioning


    • You can move the element by using tblr (top, bottom, left, right)

    • By setting the element to relative, and adjusting the top and left positions it will move the div to the specified position.

    • Relative to the parent element.

    • To bring an elemenet in front of another use   z-idex: 1;   the higher the number the greater importance.




  3. Absolute Positioning - How it is positioned relative to the parent element.

    • Boxes 2 and 3 are inside of a div with a class of container-3.


    • CSS attributes are set to move the elements within the parent div. But the parent element must be set to relative to accomplish this.


    • Now, the CSS for box-2 is set to abosolute and top/left attributes are set.


    • Box-3 CSS is then set.

  4. Fixed Positioning

    • Holds the position of the element in one place (fixed on the page)

    • This function is not working in the current demo.



  5. Sticky Positioning

    • Similar to Fixed but sticks in a certain spot.

    • Used mostly for menus where you want the menu to stick to the top.
