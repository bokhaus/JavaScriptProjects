Box Model / Margin & Padding

Notes:
  1. Basic create a link
    • Example:  <a href="#">Click here</a>
    • In this case, # represents a web address and will return the current page.
    • ***The link is blue by default and underlined

    • To change color and underline use CSS attributes


    • To change the hover color, use the psuedo selector


    • You may also increase size, weight or set other attributes on hover


    • In addition to hover there is also 'visited" which changes the link once it has been visited. Also a:active which changes the color of the link while being clicked. Neither are used frequently.



  2. Basic Button
    • Example:  <button class='btn'>My Button</button>
    • Usually uses a class such as;   class='btn'

    • Add styling to the btn class - global styling

    • cursor: pointer;  displays a pointer when hovering over a link or button

  3. Hover on button or link. Set attributes in CSS.
    • use .btn:hover {color: #f4f4f4; background: #446190;}

  4. The final version formats a link to look like a button
