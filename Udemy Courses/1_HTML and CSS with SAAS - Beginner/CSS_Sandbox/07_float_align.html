Float and Alignment

Notes:

  1. Center and set width using container element

    • EXAMPLE:  .container{ width: 960px; margin: auto; }

    • The container element wraps the div and acts on the whole.

    • margin: auto;    sets equal margin on both sides of the element

    • You may also use a measurement to set the top/bottom or left/right margin
    • EXAMPLE:  margin: 30 px auto;

    • Always use max-width: on the container. This helps when sizing the page for diffenet devices.

  2. Text alignment is by default to the "left". To align other ways, must use properties
    • text-align: right;
    • text-align: center;
    • text-align: justify;   makes the text even across the element.
    • text alignment can also be used on the div to align everything.

  3. Floats (Old School) ***Best Practice*** is "Flexbox" and/or "CSS Grid"

    • To float the element use the following code.
    • EXAMPLE:  #box-2 { float: left; width: 68%; }
    • EXAMPLE  * { box-sizing: border-box; } Removes padding - so it will need added back after spacing is correct.
    • The two need used together to maintain proper distancing.

  4. Clear Float
    • Remove the float to allow another box to be on the same level. A clear attribute must be assigned in the CSS.
    • class="clr" used inside of a div following the use of the float.
    • CSS function -  .clr {clear: both;}
